# 批量视频编码工具

## 目标

1. 带有gui的视频编码工具
2. 支持单个文件和文件夹拖放
3. 使用ffmpeg进行压缩
4. 支持自定义参数

## 需求

1. 编码参数可调整
2. 编码参数可使用自定义ffmpeg命令行
3. 输出目录可选
4. 显示进度
5. 文件夹拖放批量编码支持保留目录结构（重要需求）
   1. 如拖入的文件夹为sample
     + D:
       + TEST
         + sample
           + filesA
             + a.mkv
             + b.mkv
           + filesB
             + c.mov
             + d.mov
   2. 且输出文件夹设置为E:\output，则输出结果应为
      + E:
        + output
           + filesA
             + a.mkv
             + b.mkv
           + filesB
             + c.mov
             + d.mov
6. 支持音频流的直接复制
7. 参数包括但不限于：视频尺寸、视频编码格式、视频码率或质量、音频编码格式或仅复制流、音频码率或质量、是否复制或嵌入字母等

## 技术栈

1. 优先使用python，如有困难可检讨其它语言
2. 编码使用ffmpeg，如本机ffmpeg未在环境变量中存在，则支持指定ffmpeg所在路径
3. 需使用gui组件，用于设置编码参数，显示列表、队列和进度等
4. 使用ini或json存储设置
5. 等待补充
