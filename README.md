# 批量视频编码工具

基于PyQt5和FFmpeg的批量视频编码工具，支持拖放、批量处理、保留目录结构等功能。

## 功能特性

- ✅ 图形化界面（PyQt5）
- ✅ 支持单个文件和文件夹拖放
- ✅ 使用FFmpeg进行视频编码
- ✅ 自定义编码参数
- ✅ 保留目录结构
- ✅ 实时进度显示
- ✅ 音频流直接复制
- ✅ 配置文件保存/加载

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用说明

1. 确保系统已安装FFmpeg（或在设置中指定FFmpeg路径）
2. 运行 `python main.py` 启动程序
3. 拖放视频文件或文件夹到窗口
4. 配置编码参数
5. 选择输出目录
6. 开始编码

## 配置说明

程序会自动保存配置到 `config.json`，包括：
- FFmpeg路径
- 编码参数预设
- 输出目录
- 其他设置
